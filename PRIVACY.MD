# Privacy Policy

At OpenMemer, we prioritize your privacy. We believe in transparency and collecting the absolute minimum amount of data necessary to improve the application.

## 1. Data We Collect
We store anonymous usage data to help us understand how the tool is being used. This includes only the following technical telemetry:
- **Event Type**: The events being when the site has been visited, and if a meme has been exported. 
- **Timestamp**: When the event occurred.
- **Meme Mode**: Which mode was active (Standard, Demotivational, GIF).
- **Media Type**: Whether an image or GIF was processed.
- **GIF Mode**: Specific settings related to GIF generation (Normal mode, or Compact mode).

## 2. Data We Do NOT Collect
We explicitly **do not** collect or store any Personal Identifiable Information (PII) or content. We do not track:
- IP Addresses
- User-Agent strings (Browser details)
- Cookies
- User IDs or Device Fingerprints
- **Meme Content**: We never see or store the text you, the images you upload or memes that you export.
- Location Data

## 3. Transparency
OpenMemer is open-source. You can verify our data collection practices by reviewing the source code on [GitHub](https://github.com/r00tkitty/OpenMemer-Web).
